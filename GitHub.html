GitHub

GitHub é uma ferramenta utilizada para versionamento de arquivos e é conhecido como a grande rede social de desenvolvimento de códigos. Ele funciona como uma nuvem, que guarda o seu trabalho para que você possa utilizá-lo em qualquer lugar. Além disso, ele também é bastante conhecido por ser uma ferramenta de trabalho em equipe, assim, várias pessoas podem mexer no mesmo projeto, sem ter que copiar as versões, podendo fazer isso remotamente.

Passo a Passo

  1. Configurações Iniciais
    1. git config --global user.name "Seu nome aqui"
    2. git config --global user.email "Seu email aqui"
  2. Cd + nome da pasta (para entrar na pasta)
  3. git init
  4. git add .
  5. git commit -m "nome do commit"
  6. git remote add origin + link do repositório (que você pega no site do GitHub)
  7. git push origin master
